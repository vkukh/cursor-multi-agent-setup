## Contribute your own

Ran the prompt on a stack not listed here? Open a PR with your `.cursor/` output.

Format your example as:

```markdown
# Sample output: [Stack name]

## Project
Brief description (1-2 sentences). Size, key technologies.

## What the prompt produced
The list of files and a short description of each.

## What was good
- ...

## What needed manual adjustment
- ...

## Generated files
[Link to a gist or paste the actual files]
```

Quality over quantity — one well-documented example beats five superficial ones.
